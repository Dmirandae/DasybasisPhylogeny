Java Tool for TNT
*****************
Usage:
java jt4tnt [-options] -i <input> -o <output> -m <member>
  -i <input>    input data file in TPS format or ROW format (CoordGen)
  -o <output>   output data file in TNT format
  -m <member>   membership list file in TXT format (optional for TPS format)
  options:
    []          square brackets means optional
    -d          delete last two points (for ruler)
    -version    print version of jt4tnt
    -? -help    print this help message
Java Tool for TNT requires Java 6 or later (Java 1.6)
